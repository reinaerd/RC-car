About

Why this project?

We started this project because of an assignment for school. We had to make a smart device that would be controlled by a raspberry pi. Hence we both are real petrol heads we wanted to do something with cars. One of us had an older radio-controlled vehicle lying about in his house witch he had lost the remote, so the project was born. We wanted to make it so we could control the car remotely. But only controlling the car was not enough for the school project so we added a couple of "goals". So we came up with the idea of adding a camera, so you can see where you are going without running behind the car like a headless chicken. We also wanted to know the speed, so we would add a speed sensor and an LCD screen to read the speed on the car as well. And the furthest goal was adding an app for your phone with gyro controls. But this all together is too much out of scope, so we stuck with the camera.

Struggles

Not everything in this project went like butter. We did encounter e lot of problems as well.

- $$$ -

The biggest being the cost. We spend more than 100 euro's a part, so we are both broke now (not really). But the cost was higher than we anticipated. The biggest cost was actually the power bank because the Pi needs at least a 5V/3A output. So the power bank set us back a whopping 56 euros The best part is that we can reuse this n the future.

- burn baby burn -

Another issue was that Reinaerd is a bit clumsy and fried the light a couple of times, so we had to replace them a few times as well. Once we burnt something and could smell it but did not find where it was coming from. So we checked all the connections to find out it was the rubber fitting around one of the cables that got a bit hot after touching the RPI. So we replaced this as well.

- Nothing to see here -

But the most annoying thing to figure out was the camera. The camera seemed to be working, but we did not get an image. We downloaded the library and tried multiple solutions, but it still was not cooperating. In general, all the hardware was a bit stubborn or "tengentsjok" as we say in Bruges.
